# Compute the character frequency in a table

Read text from the input file and compute the character frequency into a dictionary.
- The computation should be case-insensitive.
- Spaces and newlines should not be counted.
- All other characters should be counted under "spl" key.

Example:
```
Hello there!

{'h': 2, 'e': 3, 'l': 2, 'o': 1, 't': 1, 'r': 1, 'spl': 1}
```

Now, write this dictionary as a table in a text file as follows. Notice that the cells are centered.

 -----------------------------------------------

 -----------------------------------------------

 -----------------------------------------------

 -----------------------------------------------

 -----------------------------------------------
|       k       |       0       |       0       |
 -----------------------------------------------
|       l       |       2       |       18      |
 -----------------------------------------------

 -----------------------------------------------

Following diagram shows the spaces left in each row:
```
If the column width is 15, a row looks like this:

1-----------------------------------------------1
|-------15------|-------15------|-------15------|
1-----------------------------------------------1

15 denotes the space between two '|' including the text written between them.
In the dividers, the 1s indicate spaces.
```
