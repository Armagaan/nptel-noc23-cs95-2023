# Parse Table

A user can provide a table in the following format:
```
 -----------------------------------------------
|    Alphabet   |   Frequency   |   Percentage  |
 -----------------------------------------------
|       a       |      173      |       8       |
 -----------------------------------------------
|       b       |       33      |       1       |
 -----------------------------------------------
|       c       |       39      |       2       |
 -----------------------------------------------
```

Write a script to read this table from the input file into a dictionary that looks like:
```python
{'a': 173, 'b': 33, 'c': 39}
```
